# Environmental Health Portfolio Website

This is an export-ready static website for Gracious Gomani.

## Files
- `index.html`
- `styles.css`
- `script.js`
- `assets/gracious-profile.jpg`

## To publish on GitHub Pages
1. Download and extract this folder.
2. Create a new GitHub repository.
3. Upload all files inside this folder.
4. In the repository settings, open **Pages**.
5. Set the source to the main branch root.
6. Save.
7. Your site will go live after GitHub finishes publishing.

## Photo
The current `assets/gracious-profile.jpg` is a placeholder. Replace it with your preferred photo using the exact same file name if you want a different image.
